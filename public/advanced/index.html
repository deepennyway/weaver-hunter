Advanced Rules

The basic rules are enough to get you going. But if you’re going to play a longer game with the same characters, you may want to add in some mechanisms to improve your character’s abilities over time. This chapter offers some options for doing that.

Note that all of these rules are optional, and are designed to add complexity to the game. Decide before you begin a new game whether and how many advanced rules your group wants to use.
